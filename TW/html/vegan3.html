    • Hummus cu sfecla rosie coapta

    INGREDIENTE

        2 bucati de sfecla medii
    400 g naut fiert
    1/2 lingurita sare
    3 linguri tahini
    2 catei de usturoi
    2 linguri zeama de lamaie
    2 linguri apa
    2 linguri ulei de masline
    2 linguri rozmarin sau chimion (optional)

    MOD DE PREPARARE

    Se spala sfecla bine si se pune la copt timp de 45 de minute. Daca atunci cand o intepati cu o scobitoare aceasta intra usor, inseamna ca este gata. Se xcoate din cuptor si se lasa la racit 5-10 minute. In timp ce aceasta se raceste, se prepara hummus-ul. Se combina toate ingredientele intr-un blender (in afara de uleiul de masline, care se adauga abia la final, treptat). Cand s-a racit sfecla,se curata, se taie bucati mici si se paseaza intr-un blender, pana e moale de tot. Se adauga peste hummus-ul facut mai devreme.
